# 科幻小说图鉴 (Science Fiction Novel Illustrated Guide)

这是一个基于 **UniApp** 和 **Vue 3** 开发的科幻小说百科全书应用。旨在为科幻爱好者提供一个探索小说世界观、角色关系、科学深度以及获取个性化推荐的综合平台。

## 🚀 项目核心功能

### 1. 智能探索与推荐
- **AI 智能推荐**：基于用户阅读历史，提供针对性的个性化小说推荐。
- **AI 小说简介**：通过集成 AI 接口，自动生成或总结小说核心内容。
- **首页聚合**：包含轮播图展示、小说主题分类展示、经典与热门作品推荐。

### 2. 深度百科 (Illustrated Encyclopedia)
- **图鉴系统**：详细记录小说中的人物、势力及其错综复杂的关系网。
- **世界观探索**：解析科幻小说中的设定、核心事件以及未来科技。
- **科学评价 (Science Rate)**：对小说中的科学设定进行硬核评估。

### 3. 社交与互动
- **讨论区**：提供用户交流心得、讨论剧情的社区空间。
- **搜索系统**：支持按关键词、标签精准查找作品。

### 4. 个人中心
- **浏览历史**：记录用户的探索足迹。
- **收藏夹**：收藏心仪的科幻作品与图鉴。
- **个人资料**：自定义用户头像与信息。

## 🛠️ 技术栈

- **前端框架**：[UniApp](https://uniapp.dcloud.net.cn/) (Vue 3 Composition API)
- **状态管理/交互**：Vue 3 `ref`, `onMounted` 等响应式特性
- **UI 组件库**：uni-icons 等 UniApp 内置组件
- **API 通信**：uni.request (对接 RESTful 接口)

## 🏁 快速开始

1. **环境准备**：
   - 安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)
   - 安装依赖：在项目根目录运行 `npm install`（如果用到第三方 Node 模块）

2. **运行项目**：
   - 在 HBuilderX 中打开本项目。
   - 点击“运行” -> “运行到内置浏览器”或“运行到小程序模拟器”。

3. **后端对接**：
   - 默认后端 API 地址位于 `api.js` 中的 `BASE_URL`。请确保本地或远程服务器已启动。
